# Gusi's Fast Weather Dock

A compact, movable, and collapsible weather control dock for Foundry Virtual Tabletop.

This module lets Game Masters quickly change the active weather effect of the current scene without opening Scene Settings, helping you keep gameplay flow fast and uninterrupted.

## Highlights

- Fast weather switching from a floating on-screen dock
- Movable and collapsible interface for flexible layout
- Lightweight workflow focused on scene control speed
- Built-in localization:
  - English
  - German

## Compatibility

- For Foundry VTT version: 14

## Installation

Install the module via Manifest URL in Foundry VTT.

1. Open Foundry VTT
2. Go to Add-on Modules
3. Click Install Module
4. Paste the module manifest URL
5. Install and enable Gusi's Fast Weather Dock for your world

## Usage

1. Enable the module in your world
2. Open a scene
3. Use the Weather Dock to select and apply weather effects quickly

The dock is designed to reduce menu navigation and speed up live scene adjustments during play.

## Development

This repository contains the source for the Foundry VTT module:

- Entry module manifest: module.json
- Main script: scripts/main.js
- Styles: styles/style.css
- Localization: lang/en.json and lang/de.json
